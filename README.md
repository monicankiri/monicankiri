# Monica Nkiri — AI Developer & System Builder

I build AI-powered systems that help businesses in West Africa stop losing customers through poor follow-up and disorganized operations.

## What I Build

I am an AI Developer building **Smartops** — an intelligent customer follow-up system for Nigerian SMEs.

logistics founders manage 30-40 merchant relationships inside personal WhatsApp with no structure, no reminders, and no system. Merchants go quiet. Nobody notices. Orders are lost silently.

Smartops fixes this. It automatically identifies which merchants need follow-up, when, and writes the message for them.

**The result:** logistics founder opens WhatsApp on Monday morning and already knows exactly who to contact and what to say.

## 🔨 Current Project — Smartops

Status:     Building in public
Stage:      Core intelligence layer (merchant risk scoring + AI message generation)
Stack:      Python · Claude API · WhatsApp Business API · FastAPI · PostgreSQL  Pandas . Sklearn . LLM APIs

Market:     Nigerian logistics SMEs (10-30 person operations)

**What it does:**
- Takes a list of merchants and their interaction history
- Scores each merchant by churn risk (days silent, complaints, order frequency)
- Generates personalised WhatsApp follow-up messages via Claude API
- Produces a weekly Monday morning report: who to contact and what to say

**What it becomes:**
- Month 4: Web dashboard — founders log in and see their at-risk list
- Month 6: Automated — system sends messages without founder touching it
- Year 2: Full SaaS product with subscription billing
- Year 3+: Platform for West African SME operations intelligence
